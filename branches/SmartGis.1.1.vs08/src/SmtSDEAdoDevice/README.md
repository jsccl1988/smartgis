# SmtSDEAdoDevice

SmartGIS ADO 数据源设备，提供基于 ADO 的数据库数据源访问功能。

## 模块简介

SmtSDEAdoDevice 是 SmartGIS 系统的 ADO 数据源设备实现，提供了基于 ADO（ActiveX Data Objects）的数据库数据源访问功能。该模块实现了空间数据引擎（SDE）接口，支持从关系型数据库中读取和写入 GIS 数据。

## 主要功能

### ADO 数据源访问
- 数据库连接管理
- SQL 查询执行
- 空间数据读写
- 事务管理

### 数据源管理
- 数据源连接
- 数据源断开
- 数据源配置
- 连接池管理

## 依赖关系

- **依赖 SmtCore**: 基础库支持
- **依赖 SmtGisCore**: GIS 数据模型
- **依赖 ADO**: ActiveX Data Objects

## 命名空间

模块使用 `Smt_SDE` 命名空间。

## 版本信息

- **版本**: 1.0
- **开发时间**: 2010-2013

# SmtSDEAdoDevice

SmartGIS ADO data source device, providing database data source access functionality based on ADO.

## Module Overview

SmtSDEAdoDevice is the ADO data source device implementation of the SmartGIS system, providing database data source access functionality based on ADO (ActiveX Data Objects). This module implements the Spatial Data Engine (SDE) interface, supporting reading and writing GIS data from relational databases.

## Key Features

### ADO Data Source Access
- Database connection management
- SQL query execution
- Spatial data read/write
- Transaction management

### Data Source Management
- Data source connection
- Data source disconnection
- Data source configuration
- Connection pool management

## Dependencies

- **Depends on SmtCore**: Base library support
- **Depends on SmtGisCore**: GIS data model
- **Depends on ADO**: ActiveX Data Objects

## Namespace

The module uses the `Smt_SDE` namespace.

## Version Information

- **Version**: 1.0
- **Development Period**: 2010-2013
